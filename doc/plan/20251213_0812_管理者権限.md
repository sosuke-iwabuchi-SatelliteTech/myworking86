# 概要
ユーザーロールを分ける

# 種類
* 利用者
    * 主にクイズ機能が使える
    * ログイン方法
        * クイズ画面で自動ログイン、ユーザー未作成の場合は初回作成時にログイン。
* 管理者
    * 裏の管理機能が使える
    * ログイン方法
        * メールアドレスとパスワードでログイン。管理者ロールのユーザーアカウントのみログインできる。

# 認可
* 利用者
    * クイズ機能が利用できる
    * 管理者機能は利用できない
* 管理者
    * クイズ機能は利用できない
    * 管理者機能が利用できる

# ポイント
* 認可外のページに移った場合はログインユーザーを切り替える。利用者から管理者の場合は管理者ログイン画面。管理者から利用者の場合はクイズトップ（自動ログイン）
