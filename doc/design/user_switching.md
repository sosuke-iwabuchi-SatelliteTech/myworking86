# ユーザー切り替え機能 (User Switching Feature)

複数のユーザー（兄弟姉妹など）が同じデバイスでアプリを使用できるように、ユーザー切り替え機能を実装しました。

## 機能概要

*   **切り替えUI**: トップ画面（WelcomeScreen）のユーザー名表示の横にあるアイコンをクリックすると、ユーザー切り替えモーダルが表示されます。
*   **ユーザー追加**: モーダル内の「あたらしくつくる」ボタンから、新規ユーザーの登録画面へ遷移できます。
*   **ユーザー削除**:
    *   現在プレイ中でないユーザーの横にゴミ箱アイコンが表示されます。
    *   アイコンをクリックすると、「本当にこのユーザーを削除しますか？」という確認ダイアログが表示されます。
    *   削除を実行すると、ユーザー情報と、そのユーザーに関連するすべてのプレイ履歴が削除されます。
    *   削除完了後、一時的に完了通知を表示し、ユーザーリストから削除されます。

## データ構造

ユーザーデータは `localStorage` に保存されます。

*   `quiz_users`: ユーザープロフィール (`UserProfile`) のリスト
*   `quiz_current_user_id`: 現在アクティブなユーザーID
*   `quiz_history_{userId}`: 各ユーザーのプレイ履歴
