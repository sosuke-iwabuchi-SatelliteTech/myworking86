# ユーザー切り替え機能 (User Switching Feature)

複数のユーザー（兄弟姉妹など）が同じデバイスでアプリを使用できるように、ユーザー切り替え機能を実装しました。

## 機能概要

*   **切り替えUI**: トップ画面（WelcomeScreen）のユーザー名表示の横にあるアイコンをクリックすると、ユーザー切り替えモーダルが表示されます。
*   **ユーザー追加**: モーダル内の「あたらしいユーザー」ボタンから、新規ユーザーの登録画面へ遷移できます。
*   **ユーザー削除**: 各ユーザーの削除ボタンから、登録情報を削除できます（確認ダイアログあり）。削除時、そのユーザーの履歴データも同時に消去されます。
