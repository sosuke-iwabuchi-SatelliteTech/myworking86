# さんすうクイズアプリ

これは、小学生向けの算数クイズを提供する、シンプルで楽しいウェブアプリケーションです。
ReactとTypeScriptで構築されており、Viteを開発サーバーおよびビルドツールとして使用しています。

## ✨ 主な機能

- **学年別のクイズ**: 1年生から4年生までの算数問題を提供します。
- **多様な問題形式**: 簡単な計算問題、九九、面積を求める図形問題など、様々なクイズが用意されています。
- **インタラクティブなUI**: 親しみやすいデザインとアニメーションで、学習を楽しくします。
- **成績の保存**: 過去10回分の成績が自動的に保存され、いつでも確認できます。
- **設定機能**: クイズ中のタイマー表示など、一部の機能をカスタマイズできます。

## 🛠️ 技術スタック

- **フレームワーク**: React 18
- **言語**: TypeScript
- **開発ツール**: Vite
- **スタイリング**: Tailwind CSS

## 🚀 セットアップと実行方法

### DevContainer を使用する場合（推奨）

VS Code と Docker がインストールされている環境であれば、DevContainer を使用してすぐに開発を始めることができます。

1.  VS Code でこのリポジトリを開きます。
2.  「Reopen in Container」という通知が表示されたらクリックします（またはコマンドパレットから `Dev Containers: Reopen in Container` を実行）。
3.  コンテナのビルドと依存関係のインストールが自動的に行われます。
4.  完了後、ターミナルで `npm run dev` を実行して開発サーバーを起動します。

### 手動でセットアップする場合

#### 必要なもの

- Node.js (v18以降を推奨)
- npm (Node.jsに同梱)

### インストール

1.  このリポジトリをクローンします:
    ```bash
    git clone https://github.com/your-username/sansuu-quiz-app.git
    cd sansuu-quiz-app
    ```

2.  依存関係をインストールします:
    ```bash
    npm install
    ```

### 開発サーバーの実行

以下のコマンドを実行すると、開発サーバーが起動します。
```bash
npm run dev
```
サーバーが起動したら、ブラウザで `http://localhost:5173` を開いてください。

## 📁 プロジェクト構成

```
.
├── src/
│   ├── components/     # Reactコンポーネント
│   ├── questions/      # クイズ問題の生成ロジック
│   ├── utils/          # ユーティリティ関数（フォーマット、ストレージなど）
│   ├── types.ts        # アプリケーション全体の型定義
│   ├── constants.ts    # 定数（学年データなど）
│   └── App.tsx         # メインのアプリケーションコンポーネント
├── public/             # 静的ファイル
└── index.html          # エントリーポイントのHTML
```

## 🧹 コード品質ツール

このプロジェクトには、コードの品質を維持するために ESLint と Prettier が導入されています。

- **Lintチェック**: `npm run lint`
- **Lint修正**: `npm run lint:fix`
- **フォーマット**: `npm run format`

コミット時に、Husky と lint-staged によって変更ファイルに対して自動的にチェックと修正が行われます。
